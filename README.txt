LEARNER PORTFOLIO TEMPLATE

FILES
-----
index.html  - The structure and content of the website
style.css   - The design, colors, spacing, and responsive layout

HOW TO OPEN THE WEBSITE
-----------------------
1. Download and unzip the folder.
2. Open the folder in Visual Studio Code.
3. Open index.html.
4. Right-click index.html and select "Open with Live Server."
5. If Live Server is unavailable, double-click index.html.

LEARNER ASSIGNMENT
------------------
1. Replace every placeholder with your own information.
2. Change the website color theme in the :root section of style.css.
3. Add your own profile image.
4. Add at least three real projects.
5. Add your LinkedIn and GitHub links.
6. Add one additional skill card.
7. Add one additional education or certification entry.
8. Make sure the website works on desktop.
9. Check spelling, grammar, and accessibility

BONUS CHALLENGES
----------------
- Add a downloadable resume.
- Add icons using Font Awesome.
- Add a dark-mode button with JavaScript.
- Add project screenshots.
- Add animations or hover effects.
